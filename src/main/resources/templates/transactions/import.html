CSV Format:
The CSV must have these columns (header row is skipped):

Description, Type, Amount, Category, Date, Notes

Type: INCOME, EXPENSE, CREDIT, or DEBIT
Date format: yyyy-MM-dd HH:mm (optional)

Example:
Freelance Payment,INCOME,500000.00,Salary,2026-01-15 09:00,January work
Office Rent,EXPENSE,200000.00,Rent,2026-01-01 00:00,Monthly rent
